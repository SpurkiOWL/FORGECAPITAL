# FORGE KAPITAL

Статический сайт для GitHub Pages, который показывает логины, колонку `Баланс с процентами` и общий баланс по данным из Google Таблицы.

## Как подключить Google Таблицу

1. Откройте Google Таблицу.
2. Создайте лист с точным названием `счета`.
3. В первой строке добавьте колонки: `логин`, `Баланс с процентами`.
4. Откройте `Расширения` -> `Apps Script`.
5. Вставьте код из `apps-script/Code.gs`.
6. Нажмите `Deploy` -> `New deployment` -> `Web app`.
7. Выберите:
   - `Execute as`: `Me`
   - `Who has access`: `Anyone`
8. Скопируйте URL веб-приложения и вставьте его в `config.js`:

```js
window.FORGE_CONFIG = {
  apiUrl: "https://script.google.com/macros/s/ВАШ_ID/exec"
};
```

## Защита оригинальной таблицы

Сайт не содержит ссылку на Google Таблицу и не содержит ID таблицы. Таблица может оставаться закрытой, а Apps Script отдаёт только поля `login` и `balanceWithPercent`.

Важно: всё, что отображается на сайте, посетитель технически может скопировать. Но перейти с сайта к оригинальной таблице без отдельной ссылки или доступа нельзя.

## GitHub Pages

Для публикации достаточно загрузить файлы в репозиторий GitHub и включить Pages для ветки `main` или `master`.
